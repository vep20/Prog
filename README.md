# Disciplina faculdade - obrigatória, 2º Semestre

**Emenda**: Uso dirigido de ferramentas para programação. Estudo de estruturas de dados básicas. Práticas de programação.

**Forma de avaliação**: 5 projetos menores e um projeto grande.

**Programa Previsto**: 
  1. Introdução ao sistema Linux. Comandos básicos
  2. Ferramentas para apoio à programação em Linux (scripts, sed, awk, dentre outros)
  3. Introdução à Linguagem C (migração de Pascal para C)
  4. Registros (struct) e Enumeração.
  5. Ponteiros e alocação dinâmica de memória, incluindo aritmética de ponteiros.
  6. Estruturas de dados fundamentais. Tipos de dados, estruturas de dados e tipos abstratos de dados.
  7. Estruturas de dados na memória principal: uso de alocação estática e dinâmica de memória.
  8. Listas lineares e suas especializações: listas ordenadas, listas encadeadas, filas e pilhas. Aplicações
  de listas.
  9. Implementação de listas, pilhas e filas em linguagem C
  10. Métodos básicos de teste de código e depuração
